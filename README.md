2. Получение Данных:

3. Исследование Данных:

4. Подготовка Данных:
   - Нормализация:              Деление всех значении на 255
   
5. Исследуемая Модель:

    - Создание **Базовой Модели**
    - Оценка **Точночти** и **Потерь**
